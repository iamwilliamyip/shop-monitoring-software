This is a software for shops or super markets to find out possible thieves. first plug off the Ethernet wire connected to your video capturing box, and connect the wire to your computer. then download the video capturing software provided by your box manufacturers on your computer, make sure you have displayed every camera channel. then launch this software to start the AI monitoring process, as the software will alarm on possible thieves, please make sure you have a speaker or sound amplifier, and turn up the volume.
